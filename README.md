# Pharmacy Management System (Java & MySQL)
A professional inventory management system for pharmacies.

### Features
- Real-time MySQL Database connectivity.
- Inventory tracking with "Low Stock" visual alerts.
- Professional sale processing with total bill calculation.
- Modern, clean GUI using Java Swing.

### How to Run
1. Ensure XAMPP or MySQL Workbench is running.
2. Create a database named `pharmacy_db`.
3. Run the following command in terminal:
   `java -cp ".;mysql-connector-j-9.7.0.jar" app.Main`
